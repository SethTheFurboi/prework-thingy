HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • img src = "PATH" creates an image, and adding alt="TEXT" simply adds context to the image
  • you can make sections to identify elements using the "ID" feature. It also helps sort your code

CSS

  • CSS is used to add style and color to certain classes of a HTML, such as the borders around these points
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A: stages your changes
  • git commit -m "UPDATETEXT": commits your changes
  • git push origin feature/add-html: pushes your changes to github
